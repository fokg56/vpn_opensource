## 🚀 Быстрый старт

### 1. Требования

- **Go 1.22+** — Установите с [go.dev](https://go.dev)
- **Веб-браузер** — Chrome, Firefox, Safari, Edge
- **Интернет** — Для установки зависимостей

### 2. Сборка

**Linux/macOS**:
```bash
cd vpn-client
bash build.sh
```

**Windows**:
```bash
cd vpn-client
build.bat
```

### 3. Запуск

```bash
./vpn-client
# Пер откройте http://localhost:8080
```

### 4. Подключение

1. Генерируем UUID: `uuidgen`
2. Генерируем ключ: `openssl rand -hex 32`
3. Заполняем форму в браузере
4. Нажимаем "Подключиться"
5. Используем прокси на `127.0.0.1:1080`

## 🎯 Основные компоненты

### 1. **Шифрование** (`internal/crypto/chacha20.go`)
- ✅ ChaCha20 поток шифра
- ✅ Генерация безопасного nonce
- ✅ Валидация ключей (32 байта)
- ✅ Потоковое шифрование/дешифрование
- ✅ Тесты и бенчмарки

### 2. **VLESS Протокол** (`internal/vless/protocol.go`)
- ✅ Кодирование/декодирование запросов
- ✅ Рукопожатие с сервером
- ✅ Поддержка IPv4, IPv6, доменов
- ✅ TCP соединения

### 3. **Менеджер туннеля** (`internal/client/tunnelmanager.go`)
- ✅ Локальный TCP слушатель
- ✅ Relay данных (bidirectional)
- ✅ Шифрование/дешифрование трафика
- ✅ Логирование в real-time
- ✅ Управление статусом

### 4. **Веб-интерфейс** (`internal/web/server.go`)
- ✅ HTTP сервер
- ✅ WebSocket для real-time логов

## 💡 Советы

### Для первого запуска:
1. Используйте адрес `127.0.0.1:8080` вместо `0.0.0.0:8080`
2. Для тестирования создайте тестовый Xray сервер
3. Проверьте что UUID совпадает с сервером

### Для развития:
1. Начните с добавления UDP поддержки
2. Затем добавьте сохранение конфигурации
3. После этого добавьте аутентификацию

1. Всегда используйте HTTPS в production
2. Валидируйте все входные данные
3. Обновляйте зависимости регулярно
