## 🚀 Как начать

### Шаг 1: Перейдите в директорию
```bash
cd /tmp/vpn-client
```

### Шаг 2: Соберите проект
```bash
# Linux/macOS
bash build.sh

# Windows
build.bat

# Или используйте Make
make build
```

### Шаг 3: Запустите
```bash
./vpn-client
# Откройте http://localhost:8080
```

### Шаг 4: Заполните параметры
- Адрес сервера: ваш Xray сервер
- Порт: обычно 443
- UUID: сгенерируйте с `uuidgen`
- Ключ: сгенерируйте с `openssl rand -hex 32`
- Локальный порт: 1080
